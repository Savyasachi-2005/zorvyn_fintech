# Zorvyn Frontend Architecture

A lightning fast Vite + React application styled elegantly using modern Tailwind CSS frameworks and visually driven through Framer Motion variants. Contains modular, role-dependent layouts scaling neatly via dynamic Context APIs.

## Local Development
1. Install dependencies aggressively:
   ```bash
   npm install
   ```
2. Launch the Vite Hot-Reload interface:
   ```bash
   npm run dev
   ```
Ensure you have mapped your `.env` file locally pointing `VITE_API_URL` back into localhost port boundaries (`http://localhost:8000/api`).
